# Redis Json + Redis OM vs 순수 redis + ObjectMapper

Redis Json은 서버가 진짜로 Json을 타입으로 이해해서 성능 차이 크게 없음. 러닝 커브만 뚫으면 전자 조합이 낫다.

10분 동안 양 측에서 채팅이 없으면 웹소켓 연결을 끊고 프론트에서도 채팅폼을 비활성화 시켜야함.

그리고 10분 이상 채팅이 없어서 채팅이 종료되었다, 10분 뒤에 채팅 내역이 사라진다 프론트에 표시하는거지.
